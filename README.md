## 🛡️ Gestão & Segurança do Hub
Para cumprir os requisitos de governança do desafio, a estrutura conta com:
* **Controle de Acesso:** Repositório configurado para auditoria e compartilhamento seguro.
* **Segurança de Credenciais:** Implementação rigorosa de arquivo `.gitignore` para mascarar chaves de API e variáveis de ambiente sensíveis.
